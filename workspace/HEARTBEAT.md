# HEARTBEAT.md

Keep heartbeats light. Do useful maintenance only when there is something real to do.

## Check

1. Check today's and yesterday's daily memory (if they exist) for `Writeback Needed`.
2. If reusable technical knowledge is pending, update `repos/wiki`.
3. If a meaningful public journey note is pending, update `repos/story`.
4. Check `git status` in `repos/wiki` and `repos/story` if they exist.
5. If nothing is due, report `HEARTBEAT_OK` and stop. Do not write a daily memory entry just to confirm a no-op heartbeat.

Do not invent work to look busy. Do not publish private or sensitive details.
